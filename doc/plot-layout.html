Plot layout definitions

The graph is divided into regions, like this:

The dimensions of these regions are defined by Options.Layout. They are specified by the following values:

  • chart height
  • chart width
  • x tick margin
  • y tick margin left
  • y tick margin right
  • plot margin
Options.Layout provides sensible defaults for these values (y tick margin right defaults to 0, for a graph with one y axis). If the graph is scaled by means of relative sizes in the <svg> element, they may not need to be changed.
